GITHUB ACTIONS - APK + AAB WORKFLOW

Added workflow:
.github/workflows/build-apk-aab.yml

What it does:
- Builds Debug APK for quick phone testing
- Builds Release AAB for Google Play Console
- Uploads both as GitHub Actions artifacts

How to run:
1. Open GitHub repository.
2. Go to Actions.
3. Select "Build APK and AAB".
4. Tap "Run workflow".
5. Wait for green check.
6. Open the completed workflow run.
7. Download:
   - WorldCupFan2026-debug-apk
   - WorldCupFan2026-release-aab

Google Play upload:
Use the .aab artifact for Play Console internal testing.
